Grok Overlay for OpenCode

A sleek, transparent desktop overlay that automatically triggers a beautiful Warp Speed particle animation when using Grok through OpenCode.

Read the full guide View on GitHub

✨ Features

Auto show / hide

Animation appears when Grok is generating and fades away when idle.

Warp Speed particles

Smooth Canvas field plus 10 presets: Hyperspace, Datastream, Orbit, Aurora, and more.

Transparent & draggable

No chrome while generating — pure ambient overlay on your desktop.

No extra API key

Works with your existing Grok / OpenCode setup and Premium access.

Real-time bridge

Python monitor for generation lifecycle + SQLite token polling.

Token flow

Floating +N chips ride the stream as tokens increase; per-style alpha controls.

🛠 Tech stack

  • Tauri 2
  • Svelte 5
  • Rust
  • Python bridge
  • Canvas 2D particles
  • Windows & macOS

⚠️ Actively developed. Contributions and feedback are welcome on GitHub.

Quick start

Windows

Build overlay → install-shim.ps1 → install-autostart.ps1 → run opencode.

macOS

Build .app → ./install-shim.sh → ./start-overlay.sh → run opencode.
